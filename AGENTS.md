## Project Summary
A premium streetwear brand website called "Memories Division", featuring a minimalist, luxury urban aesthetic with smooth animations and a full e-commerce experience.

## Tech Stack
- Framework: Next.js 15 (App Router)
- Styling: Tailwind CSS
- Animations: Framer Motion
- Icons: Lucide React
- Components: Shadcn UI (Radix UI)
- State Management: React Context (for Cart)
- Deployment: Vercel (recommended)

## Architecture
- `src/app`: Page routes and layouts
- `src/components`: Reusable UI components
- `src/hooks`: Custom React hooks
- `src/lib`: Utility functions and constants
- `src/context`: React Context providers (Cart)

## User Preferences
- Clean, minimal, urban, stylish design
- Black, white, gray with one accent color (Electric Blue suggested)
- Smooth scroll and hover animations
- Dark mode support

## Project Guidelines
- Use functional components with TypeScript
- No comments unless requested
- Responsive design for all screen sizes
- Performance and SEO optimized

## Common Patterns
- Framer Motion for entrance and scroll animations
- Tailwind for responsive styling
- Lucide React for consistent iconography
